Mensagens

Utilizadas para prover a comunicação entre objetos

A interação entre os objetos é feita através das mensagens, que são enviadas de um objeto a outro, solicitando a execução de um método

Mensagens

Definem:

  • Nome do serviço requisitado
  • Informação necessária para a execução do serviço
  • Nome do requisitante

Mensagens

Na prática são implementadas como métodos

  • Nome do serviço requisitado: identifica o método
  • Informação necessária para a execução do serviço: lista de parâmetros do método
  • Nome do requisitante: objeto que realizou a chamada!

Métodos

  • São procedimentos ou funções que manipulam os dados
  • Realizam as ações que aquele objeto pode executar

Métodos


class NomeClasse {
    tipo atributo_1;
    tipo atributo_2;
    tipoRetorno nomeMetodo1(parametros){
	//faz alguma coisa
        return ...;
    }
    tipoRetorno nomeMetodo2(parametros){
	//faz alguma outra coisa
        return ...;
    }
}

Método Construtor

  • Não é possível invocar um método de uma classe sem criar um objeto dela
  • O método construtor instância (cria um objeto) de uma classe
  • Aloca recursos necessários
  • Inicializa atributos

Método Construtor

  • Não possui tipo de retorno
  • Possui o mesmo nome da classe

class NomeClasse {
    NomeClasse(){
	//método construtor
	...
    }
    tipoRetorno nomeMetodo1(parametros){
	//faz alguma coisa
        return ...;
    }
}

Método Construtor

  • Toda classe tem no mínimo um método construtor
    • Caso o programador não escreva um método construtor, o compilador adiciona um método default a classe, sem argumentos e sem corpo
  • Classes podem ter mais de um método construtor
    • Listas de parâmetros distintas

Método Main

  • Método que inicia a execução do programa
  • Cada classe pode possuir um método main
  • Qual main será executado?
    • Definido no momento da execução

Método Main


class NomeClasse {
    NomeClasse(){
	//método construtor
	...
    }

    public static void main(String args[]){
	//a execução do programa começa daqui
    }
}

Exemplo


class Quadrado {
    int tamanhoDoLado;
    Quadrado(int l){
       tamanhoDoLado = l;
    }
    Quadrado(){
    
    }
    int area(){
        return tamanhoDoLado*tamanhoDoLado;
    }
    int perimetro(){
	return tamanhoDoLado*4;
    }
    public static void main(String args[]){
	Quadrado q = new Quadrado(10);
	Quadrado q2 = new Quadrado();
	System.out.println(q.area());
	System.out.println(q.perimetro());
	System.out.println(q2.area());
	System.out.println(q2.perimetro());
    }
}

Exercício 1

Escreva uma classe que represente uma conta bancária

  • A conta armazena o nome, cpf e saldo do cliente
  • Deve ser possível operações de saque, depósito e visualização de saldo
  • Não deve ser permitido o saque em contas sem saldo

Exercício 2

Escreva uma classe que represente um ponto no plano cartesiano

  • Coordenadas X e Y
  • Métodos para inicializar e calcular a distância até outro ponto

Exercício 3 (entregar)

Escreva uma classe que represente um Círculo

  • Atributos: nome, raio e centro (objeto da classe Ponto)
  • Métodos
    • Inicializar os dados
    • Calcular diâmetro
    • Calcular área
    • Calcular circunferência
    • Modificar nome
    • Exibir os dados
